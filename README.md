# AWS Infrastructure Cost Optimization

This project demonstrates AWS infrastructure cost optimization using EC2, Auto Scaling, CloudWatch, and S3 lifecycle management.

## Services Used
- Amazon EC2
- AWS CloudWatch
- Auto Scaling
- S3
- Apache HTTP Server

## Features
- Live website hosted on EC2
- Monitoring setup
- Secure inbound rules
- Professional dashboard UI
